    Important issue - the 0 Ohm resistance can't be remove!

            I have tried so many times and found that if the 0 Ohm resistance been remove, 
        the time you use it to make fuse/make program to other programmer, 
        it must fail!
        
    And if you don't remove the 0 Ohm resistance
    
        Everything will be OK!
        
        
        I ask this this with sending the global email ( I wrongly thought it may be a big mistake written in the tutorial )
        Then I got the reply of this problem from Eduardo Chamorro Martin, Fablab Seoul:
        
    Removing that 0 resistor allows you to program other
     boards with voltage other than the 5v coming from the USB.
(if you remove that resistor,you are not
     powering the "target" board) that means you have to power the
     "target" board with his own power supply.
    

If you only want to use it to program 5v targets, you can keep
     it and power the "target" with the same flat ribbon cable.
